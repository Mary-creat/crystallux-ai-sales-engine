# Crystallux MGA Operations

Documentation and operational materials for Crystallux MGA (separate legal entity from the Crystallux Platform).

This directory will house:
- Sub-agent contracts (templates)
- Compliance checklists (FSRA, CASL)
- Carrier contact records
- Recruitment playbooks
- Onboarding runbooks
- Production tracking reports

These materials may migrate to a separate repository (crystallux-mga-ops) once MGA operations formalize beyond Month 3. See BUSINESS_PLAN.md section 5 and ARCHITECTURE_DOCTRINE.md for context on why MGA is structured as a separate business line from the Platform.
