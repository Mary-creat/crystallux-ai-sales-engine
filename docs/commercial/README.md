## Principles applied across every file in this folder

- **Peer-advisor tone.** Never a vendor pitch. Never a hype deck.
- **Canadian context.** CASL compliance in outreach, PIPEDA in data handling, GST/HST in pricing, Ontario jurisdiction.
- **Concrete numbers.** Every claim has a reproducible target. No "up to 10×" or "unlimited growth."
- **Copy-ready output.** Mary does not need to draft from outline. She customises names, URLs, and dates.
- **Designer-aware.** Handoff notes for every document that touches a designer's workflow.
