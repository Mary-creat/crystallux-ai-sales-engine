"Let's make a deal!" 24/7 live-auction host. Dashboard in progress.
What LUXI will surface here
  1. Open auctions — live rows from auctions with current high-bid, anti-snipe state, scheduled close.
  2. Bidder trust tiers — the 4-tier anti-fraud state machine (bidder_trust_scores). How many bidders at each tier, fraud signals.
  3. Payment holds — outstanding Stripe PaymentIntent authorizations (auction_payment_holds) and capture status.
  4. Streaming sessions — past + scheduled multi-platform broadcasts (avatar_streaming_sessions). Runtime decision pending LIVE_STREAMING_RFC.md.
  5. Operator controls — pause/resume auction, manual forfeit, ban a bidder, force-release a hold.

The schema is in place from db/migrations/avatars-platform-schema.sql. The auction tick workflow (T1.7) + streaming runtime are the next builds. LUXI Phase 2 (consignment + inventory + comment-checkout + Stripe Connect for sellers) is its own multi-tranche workstream.
