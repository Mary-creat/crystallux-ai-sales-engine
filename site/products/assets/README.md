# Product page assets

Drop image files here. They're served at `/products/assets/<filename>`.

## Your founder photo
Save a square photo of yourself as **`mary.jpg`** in this folder.
It appears automatically in the founder section on the Sentinel and
Sales Engine pages. Until the file exists, those pages show your
initial ("M") in a circle instead, so nothing ever looks broken.

- Shape: square (it's cropped to a circle). ~400×400px is plenty.
- Name it exactly `mary.jpg` (lowercase).

## Real product screenshots (optional, even better)
You can also drop real dashboard screenshots here and swap them into the
product pages in place of the CSS console mockups. Suggested names:
- `sentinel-console.png`
- `sales-engine-leads.png`

The HTML for each page has a comment showing the one line to paste.
